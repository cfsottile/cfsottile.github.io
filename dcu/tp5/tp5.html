Interfaz con signos de adaptación: Mozilla Firefox

El navegador web Mozilla Firefox presenta una interfaz adaptable, ya que es el usuario quien modifica a la interfaz de forma activa interactuando con el sistema. Se permite adaptar:

  • Apariencia de la interfaz: fuente, tamaño de la fuente, colores de fuentes y enlaces, imágenes de fondo, colores para las barras de herramientas.
  • Contenido de la interfaz: valores por defecto (como dónde guardar las descargas y qué página abrir al inicio), agregación/eliminación y disposición personalizada de íconos.
  • Comportamiento de la interfaz: mediante la instalación de plugins, se pueden agregar combinaciones de teclas para realizar distintos tipos de acciones, así como crear atajos.

El propio usuario decide cuándo efectuar cambios sobre la interfaz, accediendo a menús de configuración. Permite tener una cuenta en la que podrían guardarse todos los datos de personalización.

Interfaz colaborativa: Codeshare

Codeshare es una aplicación web sencilla que consiste en un editor de código con soporte para la sintaxis de diferentes lenguajes, y posibilidad de que haya múltiples usuarios conectados, los cuales podrán ver los cambios que todos realicen en tiempo real.

Coordinación

Los usuarios pueden crear diferentes espacios, cada uno de los cuales se comparte mediante un enlace. Un espacio consiste en un buffer único de texto; no hay noción de archivos ni directorios. El creador de cada espacio puede elegir que aquellos con quienes comparta el enlace, puedan sólo ver y no editar.

Comunicación

La única herramienta de comunicación provista es un chat de video.

Colaboración

Al ingresar a un enlace, cada usuario ve el mismo espacio. Dentro de un espacio se pueden efectuar modificaciones en el texto, y cada cambio realizado es visualizado por todos. Además se puede resaltar código, y a los otros usuarios también les figurará ese código resaltado.

Otros

Lo único privado es la personalización de la interfaz (temas y atajos de teclado). Luego, la edición del código es en su totalidad pública.

CodeBunk: un editor colaborativo más avanzado

La sencillez de Codeshare permite que en sólo un par de clics tengamos disponible un editor de código con un espacio para compartir y sobre el cual colaborar en tiempo real, con posibilidad de comunicarnos vía chat de video. Esto servirá para poder editar código entre múltiples usuarios, pero no será suficiente si queremos, por ejemplo, compilar y correr el código dentro de la misma aplicación, o iniciar un chat de texto. Estas últimas características son agregadas en la aplicación web CodeBunk: un REPL en el que probar código y donde podremos cargar y/o correr el que estamos editando, y un espacio de chat de texto.

Interfaz telefónica: Atención al cliente de Personal (*111)

El sistema de atención al cliente de Personal permite efectuar diversos trámites, como consultar números de pago, efectuar pagos, informar pagos, efectuar recargas de saldo, informar robos o pérdidas de equipos, inicar el trámite de portabilidad, acceder a descuentos en el cambio de equipos, adquirir productos, conocer beneficios y novedades, e informar cambios en la línea. También ofrece comunicarse con un representante, para continuar la atención de forma personalizada. La interacción se lleva a cabo mediante los botones del teclado, no proveyendo reconocimiento de voz ni sugerencias.
